# Pixiv Scrapper


Library for scrapping www.pixiv.net pages 

## Installation

Add the dependency on your `build.gradle.kts`

```kotlin
dependencies {
    implementation("io.github.magonxesp:pixiv-scrapper:0.0.0")
}
```
